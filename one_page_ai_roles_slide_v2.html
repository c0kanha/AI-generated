Most roles think AI is coming to replace them.

Business: “AI can write my docs, plans, and presentations... so even my meetings may become optional.”
Developers: “AI can generate my code... so typing fast is no longer a superpower.”
SMEs: “AI can learn the business rules in my head... so my ‘only I know this’ advantage feels shaky.”
QA: “AI can create test cases and catch bugs faster... so even my defect list is not safe.”
Technical Architects: “AI can suggest system designs and integration patterns... so high-level thinking no longer feels untouchable.”
PMs / BAs: “AI can turn meetings into requirements, tasks, and updates... so even status calls are starting to look dangerous.”
Incident Teams: “AI can read tickets, spot patterns, suggest root causes, and draft resolutions... so even production panic now has competition.”
Support Teams: “AI can answer the same questions instantly, politely, and 24/7... so patience is no longer a job skill.”
Operations Teams: “AI can monitor systems, raise alerts, and handle routine issues automatically... so night support may sleep before we do.”
Designers: “AI can generate screens, flows, and variations in seconds... so blank canvases are scared too.”
